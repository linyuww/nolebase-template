每个位置上跳跃的距离 `<=`该元素大小



0 + x .... >= size - 1

找最后一个能到终点的

倒推？

第一次

```c++
    bool canJump(vector<int>& nums) {
        int j = nums.size() - 1;
        for(int i = j - 1;i >= 0;i--){
            if(nums[i] + i >= j){
                if(i == 0)return true;
                j = i;
            }
        }
        return false;
    }
```
